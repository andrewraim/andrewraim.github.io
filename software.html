R packages

  • sfreader: An R Package to Read Census Summary Files. Currently focused on the 2010 SF2 summary file.
  • DirectSampling: Direct sampling with a step function. Also see DirectSampling.jl for a Julia implementation.
  • COMMultReg: Conway-Maxwell multinomial regression.
  • allocation: Exact optimal allocation algorithms for stratified sampling.
  • stcos: Space-time change of support.
  • OverdispersionModelsInR: Some tools to help with MLE computation, especially for non-standard likelihoods found in overdispersion models.
  • LDR: Methods for likelihood-based dimension reduction in regression
  • ManifoldOptim: An R interface to the ROPTLIB library for Riemannian manifold optimization.
  • MixLink: Mixture link regression.
  • COMPoissonReg: Conway-Maxwell Poisson (COM-Poisson) regression.

LaTeX templates

  • latex-resume: a template for resumes/CVs.
  • latex-business-card: a template for business cards.
  • beamerthemeraim: a custom Beamer theme.
  • formula-sheet: a template for formula sheets.
  • sweave-examples: templates for Latex slides and articles using Sweave via knitr.

Note: I am not a LaTeX programmer, so there may be room for improvement in these templates.

Utilities

  • md-util: some command line scripts and templates to facilitate compiling markdown to various formats (PDF, html, Beamer) using Pandoc.
  • worker: a tool to help run repetitive computational studies such as statistical simulations.
